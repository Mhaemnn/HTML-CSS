Konsep Dasar:

  • setiap element dihalaman berada dalam sebauah box(kotak)
  • kita bisa mengatur ukuran dan posisi kotak tersebut
  • kita bisa memeberi warna dan gambar sebagai background kotak tersebut.

  1. Margin: area transfarand disekitar kotak
  2. Padding: area transfaran di dalam kotak
  3. border: batas disekeliling content dan Padding
  4. content: sebenernya didalam box, bisa teks atau gambar
